# Samara-University-HPC-Spring-2026
Настоящий курс по дисциплине "Высокопроизводительные вычисления" предназначен для студентов магистерской программы ПМИ Самарского университета, весенний семестр 2026.

Лектор: [Якимов Павел Юрьевич](https://ssau.ru/staff/222993132-yakimov-pavel-yurevich) 
- доцент кафедры технической кибернетики, к.т.н.
- директор ООО "Центр информационных технологий" 

При подготовке курса были использованы следующие материалы:
- [NVIDIA CUDA C++ Programming Guide](https://docs.nvidia.com/cuda/cuda-c-programming-guide/index.html)
- [CUDA C++ Best Practices Guide](https://docs.nvidia.com/cuda/cuda-c-best-practices-guide/index.html)
- Материалы компании Applied Parallel Computing LLC
- Наработки автора
